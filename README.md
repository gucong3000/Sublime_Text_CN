## 汉化 ##

汉化前请先安好所需插件

1. 安装 [Node.js](https://nodejs.org/) 并 copy `Han.js`与`package.json`到Sublime Text安装目录
1. 在管理员权限下，命令行运行 `npm i` 再运行 `node Han.js`

> 新装插件后造成界面部分英文化，可再次运行`node Han.js`，即可恢复正常。
